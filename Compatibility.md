# Compatibility

## Linux

Trunk는 Ubuntu, Arch 등을 포함한 대부분의 Linux 버전에서 실행됩니다. 단, glibc 버전 2.19 이상이 필요합니다. 알파인 리눅스는 지원되지 않습니다.

## macOS

Trunk는 macOS 버전 10.15 이상에서 실행됩니다.

## Windows

[현재 공개 베타 버전입니다.](<./Windows%20Support%20(beta).md>)

## Backward Compatibility

일반적으로 [Trunk Launcher](./Components.md#trunk-launcher)와 Trunk binary 간의 하위 호환성을 유지하기 위해 노력하고 있지만, 최신 버전의 Trunk를 지원하기 위해 가끔 런처를 업그레이드해야 할 수도 있습니다.
